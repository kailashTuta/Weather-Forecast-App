# Project Prompts Collection

This folder contains all the prompts used during the Weather Forecast App migration project from AngularJS to React.

## Available Prompts

### 01. Ontology Schema Generation
**File:** `01-ontology-schema-generation.md`

Initial prompt for creating the comprehensive JSON-LD schema that defines the migration ontology, including all components, requirements, stakeholders, and relationships.

### 02. Requirements Documentation
**File:** `02-requirements-documentation.md`

Prompt used to generate detailed documentation for functional and non-functional requirements based on the ontology schema. Created comprehensive markdown files in the docs folder.

### 03. Context Studio MCP Server Query
**File:** `03-context-studio-mcp-query.md`

Prompt for listing and retrieving all available requirements from the Context Studio MCP (Model Context Protocol) server using a specific context ID. Demonstrates integration with external requirement management systems.

## How to Add More Prompts

If you have additional prompts from your chat history, please add them to this folder using the following structure:

### File Naming Convention
- `XX-descriptive-name.md` (where XX is a sequence number)

### Prompt Template

```markdown

## Date
[When this prompt was used]

## Prompt
[Your actual prompt text here]

## Context
[Why you needed this prompt, what problem it solved]

## Result
[Brief description of what was accomplished]

## Related Files
- List of files created or modified
```

## Categories for Organization

Consider organizing prompts by:
- **Setup & Configuration** - Initial project setup
- **Migration Planning** - Strategy and architecture decisions
- **Component Development** - Creating React components
- **Bug Fixes** - Troubleshooting and fixes
- **Features** - New feature implementations
- **Documentation** - Creating docs and guides
- **Testing** - Test creation and QA
- **Deployment** - DevOps and deployment
- **Optimization** - Performance improvements
